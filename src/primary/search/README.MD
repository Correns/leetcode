# 顺序查找
## In order to find
对数组进行遍历比较查询的基本算法

```时间复杂度：O(n)```
# 二分查找
## Binary Search
折半查找，要求线性表必须采用顺序存储结构，而且表中元素按关键字有序排列。

```时间复杂度：O(log2n)```
